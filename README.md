#  Mini Web Security Scanner Dashboard

A full-stack web application that scans websites for common security vulnerabilities and displays results in an interactive dashboard.

---

##  Features

-  Scan any URL for vulnerabilities
-  Detect:
  - Cross-Site Scripting (XSS)
  - SQL Injection (basic detection)
  - Missing Security Headers
-  Risk Scoring System (Low, Medium, High)
-  Interactive Dashboard (React)
-  Backend Scanner Engine (Python)

---

##  Tech Stack

### Frontend
- React.js
- CSS

### Backend
- Python (Flask)
- Requests Library
